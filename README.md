## 中田

・git resetの使い方一覧
git reset HEAD <ファイル名>	ステージング登録取り消し
git reset --hard HEAD^	直前のコミットを取り消す(ワークディレクトリの内容を書き換える)
git reset --soft HEAD^	直前のコミットを取り消す(ワークディレクトリの内容をそのまま)

・HEADについて
HEAD^　直前コミット
HEAD~{n}　n個前のコミット
