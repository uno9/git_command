1. ailasについて

ailasは便利なので絶対設定してください！！
checkout, branchやcommitなどの文字を省略できる機能です。

設定前: git checkout master
設定後: git ck master

設定前: git branch -D test
設定後: git br -D test

設定の仕方は色々ありますが、下記のサイトではワンラインで済むものがまとめられています。
https://dev.classmethod.jp/articles/git-config-alias-19/
